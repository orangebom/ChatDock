# ChatDock

一个基于 `Tauri + WebView2` 的 Windows 桌面工具。在一个窗口内同时展示 `ChatGPT`、`Gemini`、`DeepSeek`、`Kimi` 四个 AI 页面，并尝试把同一个问题同时发送给它们。

## 当前能力

- 单窗口四宫格展示 4 个 AI
- 每个 AI 使用独立持久化 `WebView` 数据目录
- 首次登录后，后续启动通常可复用登录状态
- 顶部统一输入框，一次触发 4 个页面发送
- 支持刷新 4 个页面和重新计算布局

## 技术说明

- Windows 下 Tauri 底层使用 `Edge WebView2`
- 不是控制 Chrome 本体，而是自建桌面浏览器容器
- 登录状态依赖各自的本地 `dataDirectory`

## 运行

```powershell
npm install
npm run dev
```

## 首次使用

1. 启动应用
2. 在四个面板里分别完成登录
3. 后续再次打开应用时，会尝试复用本地登录状态

## 已知限制

- 由于不是 API，而是直接操作网页 DOM，页面结构变化时可能需要更新选择器
- 某些站点在特定场景下可能触发风控、验证码或重新登录
- “同时发送”是近乎同时，不是毫秒级严格同步
- 如果某个站点改版较大，当前发送脚本可能失效
